1. 历史消息记录 :messages :mes message-history

":messages" 命令可查看过往输出的消息。适合重新检视可能被覆盖或截短的历史消息。 消息具体的显示方式取决于 'shortmess' 选项。 :mes[sages] 显示所有历史消息。 :{count}mes[sages] 显示最新 {count} 条消息。 :mes[sages] clear 清空所有历史消息。 :{count}mes[sages] clear 清空旧消息，仅保留最新 {count} 条消息。 保存的消息总数由 'messagesopt' 选项控制。 g< 本命令用于察看上一条命令输出的最后一页内容。在提示 "请按回车继续" 时，容易误按 <Space> 翻页跳过有用信息。本命令可回到按回车继续的提示画面，然后可向上翻页查看 更早输出。 注意: 在提示显示更多信息时如果按 "q" 提前退出，仅能查看退出前的内容。 执行新命令产生输出时，旧命令的输出缓存会被清空。 本命令的输出无法被重定向捕获。 使用翻译消息 (如中文消息) 时，首行会显示消息或翻译维护者信息。发现问题可据此联 系维护者。 要查找特定 (提示或错误) 消息，可用消息开头的 ID 编号作为帮助主题。例如，要获取 以下消息的帮助: E72: Close error on swap file 或 (翻译版本): E72: 交换文件关闭错误 可用: :help E72 大小写不敏感: :help e72 编号本身并无特殊含义，仅用作唯一标识。
